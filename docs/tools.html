Tools

The capabilities that agents use to interact with your system.

Overview

Tools are the hands of C-napse. When an agent decides on an action, it executes one or more tools to accomplish it. All tool executions are logged and can be reviewed.

Safety: Tools that modify files or execute commands will prompt for confirmation by default. Use --yes to auto-approve.

File System Tools

Tool Description
read_file Read file contents
write_file Create or overwrite a file
append_file Append content to a file
list_dir List directory contents
search_files Search for files by name/pattern
search_content Search file contents (grep)
move_file Move or rename files
copy_file Copy files
delete_file Delete files (with confirmation)

Shell Tools

Tool Description
execute Run a shell command
execute_background Run command in background

Example

// Agent executes:
execute("git status")

// Returns:
{
  "stdout": "On branch main\nnothing to commit",
  "stderr": "",
  "exit_code": 0
}

Process Tools

Tool Description
list_processes List running processes
process_info Get details about a process
kill_process Terminate a process
system_info Get system information

Web Tools

Tool Description
http_get Make HTTP GET request
http_post Make HTTP POST request
download_file Download file from URL
