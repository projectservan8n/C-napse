Agent System

C-napse uses a multi-agent architecture where specialized agents handle different types of tasks. A router agent analyzes your query and delegates to the most appropriate specialist.

Specialized Agents

Router Agent

Query classification

The brain of the system. Analyzes queries to determine intent, selects the appropriate specialist, and extracts relevant context.

  • Intent classification
  • Agent selection
  • Context extraction

Shell Agent

System operations

Handles system operations, file management, and command execution. The hands of C-napse for interacting with your system.

  • Execute shell commands
  • File operations
  • Process management

Code Agent

Code assistance

Specializes in code understanding, generation, and modification. Your pair programmer for complex coding tasks.

  • Code explanation
  • Bug identification
  • Code generation

Research Agent

Information retrieval

Handles information gathering and knowledge synthesis. Answers questions and explains complex concepts.

  • Answer questions
  • Explain concepts
  • Compare technologies

Force Agent Selection

While the router automatically selects the best agent, you can force a specific agent:

Terminal 
# Force shell agent
cnapse -a shell "list all rust files"

# Force code agent
cnapse -a code "explain this function"

# Force research agent
cnapse -a research "what is the difference between TCP and UDP?"
Best Practice

Let the router choose automatically for most tasks. Only force an agent when you know the router is making incorrect selections.
