Agent System

C-napse v0.10.0 features a truly autonomous multi-agent architecture. The core Autonomous Agent uses a think-observe-act loop with up to 25 attempts, self-learning memory, and multi-LLM help when stuck.

Specialized Agents

Computer Agent

Desktop automation

Controls your desktop like a human would. Mouse movements, keyboard input, window management, and app launching.

  • Mouse and keyboard control
  • Window management
  • App launching via Run dialog

Browser Agent

Web automation

Browser automation via system commands + keyboard/mouse control. Navigate websites, fill forms, and interact with web apps using your default browser.

  • Web search (Google, etc.)
  • Email via Gmail/Outlook
  • Google Sheets/Docs

Vision Agent

Screen analysis + element finding

Captures screenshots and uses AI vision to understand what's on screen. Extracts element coordinates for clicking. Supports Ollama VLMs, Claude, GPT-4 Vision.

  • Find element coordinates on screen
  • Screen change detection (hash-based)
  • Background screen monitor service

Autonomous Agent

25 attempts, self-learning

Persistent think-observe-act loop with up to 25 attempts. Asks Perplexity, ChatGPT, or Claude.ai for help when stuck. Learns from successes.

  • Multi-LLM help (Perplexity, Claude, ChatGPT)
  • Memory persistence (~/.cnapse/agent-memory.json)
  • Screen change detection and verification

Learner Agent

Memory & pattern recall

Self-learning system that saves successful actions to persistent memory. Recalls what worked before when facing similar situations.

  • Save successful action patterns
  • Fuzzy matching for recall
  • Success/failure tracking

Shell Agent

System operations

Handles system operations, file management, and command execution. The hands of C-napse for interacting with your system.

  • Execute shell commands
  • File operations
  • Process management

Telegram Agent

Remote + streaming

Control your PC from anywhere via Telegram. New commands: /agent for autonomous goals, /watch for live screen streaming, /learn for memory.

  • /agent <goal> — autonomous task execution
  • /watch — live screen streaming
  • /learn — view learned actions

Force Agent Selection

While the router automatically selects the best agent, you can force a specific agent:

Terminal 
# Force shell agent
cnapse -a shell "list all rust files"

# Force code agent
cnapse -a code "explain this function"

# Force research agent
cnapse -a research "what is the difference between TCP and UDP?"
Best Practice

Let the router choose automatically for most tasks. Only force an agent when you know the router is making incorrect selections.
