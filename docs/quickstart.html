Quick Start

Get C-napse running on your system in under 2 minutes. This guide covers installation, initial configuration, and running your first autonomous task.

Requirements

Operating System

Windows 10+, macOS, Linux

Memory

4GB RAM minimum

Network

Internet for API providers

Storage

50MB for binaries

Installation

1

Run the installer

Open your terminal and run the appropriate command for your operating system:

PowerShell (Windows) 
irm https://raw.githubusercontent.com/projectservan8n/C-napse/main/install.ps1 | iex
Bash (macOS / Linux) 
curl -fsSL https://raw.githubusercontent.com/projectservan8n/C-napse/main/install.sh | bash
2

Verify installation

Confirm C-napse is installed correctly by checking the version:

Terminal 
cnapse --version
Expected Output

You should see cnapse 0.1.0 or similar version number.

3

Configure your API provider

C-napse needs an LLM provider to function. Set up your preferred provider:

config.toml 
# ~/.cnapse/config.toml

[llm]
provider = "anthropic"  # or "openai", "ollama", "openrouter"
model = "claude-3-5-sonnet-20241022"

[llm.anthropic]
api_key = "sk-ant-..."  # Set via ANTHROPIC_API_KEY env var

[llm.openrouter]
api_key = "sk-or-..."  # Set via OPENROUTER_API_KEY env var
Security Note

Never commit API keys to version control. Use environment variables instead.

Your First Task

Launch C-napse and try your first autonomous task:

Terminal 
# Start C-napse
cnapse

# Or start with a specific task
cnapse "List all files in the current directory and summarize their purpose"

Example Tasks

Here are some things you can ask C-napse to do:

  • File operations: "Create a new folder called 'projects' and organize my downloads by file type"
  • Code analysis: "Analyze this codebase and explain the architecture"
  • System tasks: "Check my system's resource usage and report on memory usage"
  • Research: "Search for the latest news about Rust programming and summarize the key points"
Pro Tip

Use the --verbose flag to see detailed reasoning and tool usage during task execution.
